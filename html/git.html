Git

  • git -v -Эта команда выводит версию Git, если он правильно установлен.
  • git init - Эта команда инициализирует репозиторий.
  • git clone - Эта команда клонирует удаленный репозиторий с GitHub себе на компьютер.
  • git status - Эта команда показывает текущий статус изменений в файлах.
  • git add - Эта команда добавляет отслеживаемые файлы.
  • git commit - Эта команда создает комит.
  • git push - Эта команда вносит изменения в удаленный репозиторий.
